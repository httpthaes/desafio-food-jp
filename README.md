## 📌 Sobre o Projeto

Este projeto faz parte dos [Desafios Frontend](https://github.com/iuricode/desafios-frontend) da comunidade **CodeLab**, criados por [@iuricode](https://github.com/iuricode). 
O objetivo é reproduzir um layout pronto no Figma em HTML, CSS e JavaScript — simulando o fluxo real de trabalho de um desenvolvedor front-end. 
O desafio #19 é o **FoodJP**, uma landing page de restaurante japonês.

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica da página |
| CSS3 | Layout (Flexbox e Grid), animações, design responsivo |
| JavaScript (ES6+) | Renderização dinâmica dos cards de comentários |
| Google Fonts | *Poppins* (corpo) + *Koh Santepheap* (títulos) |
